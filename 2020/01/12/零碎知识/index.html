零碎知识

复合类型

  • 一个基本数据类型 + 一组声明符（声明符包含类型修饰符 * 和 &）

  • 复杂的指针和引用可以从右向左读，离变量名最近的的类型修饰符对变量的类型产生最直接的影响，声明符的其他部分用以确定r引用的类型是什么

  • 拷贝一个对象的值不会改变他，拷贝完成，新对象与原对象无关

  • 一个文件中给出定义，多个文件中可以声明（用关键字extern，而且不能重复定义）

  • 默认状态下，const 对象仅在文件内有效（在定义和声明时都用const就可以都有效）

  • 引用只能绑定在对象上，不能与某个字面值或表达式的结果绑定，有两个例外

    第一个就是： const int &r1 = 42;(表达式能转化为引用的类型）

constexpr 和 常量表达式

  • 常量表达式指值不会改变并且在编译过程中就能得到计算结果的表达式，字面值属于常量表达式 。只有本身是常量，且初始值是常量表达式时，才是常量表达式。
  • 将 变量声明为 constexpr 以便编译器验证变量的值是否是一个常量表达式，声明为 constexpr 的变量一定是常量，必须用常量表达式初始化。
  • 总之，常量表达式一定是在编译时就能计算出值的常量，可以用constexpr函数初始化constexpr常量
  • constexpr把他所定义的对象置为了顶层const
  • 如果你认为他是常量就给他定义为constexpr
